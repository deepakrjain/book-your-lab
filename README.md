# Book Your Lab

A modern and responsive website for booking lab tests and diagnostic services online. Built with HTML, CSS, and JavaScript.

## Features

- Clean and intuitive user interface
- Responsive design that works on all devices
- Easy test booking and search functionality
- Health package browsing
- Popular tests and health checkup packages
- Newsletter subscription
- Contact and location information
