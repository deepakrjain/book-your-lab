# Book Your Lab

A website for booking lab tests and diagnostic services online with clean ui. Built with HTML, CSS, and JavaScript.

## Features

- Clean and intuitive user interface
- Responsive design that works on all devices
- Easy test booking and search functionality
- Health package browsing
- Popular tests and health checkup packages
- Newsletter subscription
- Contact and location information

## Live Demo

Visit the live website: https://book-your-lab-woad.vercel.app

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons

## Getting Started

Simply open `index.html` in your web browser to view the website locally.
